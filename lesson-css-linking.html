5. CSSの読み込みと適用ルール

← カリキュラム一覧へ

CSSが反映されるまで

  1. HTMLの <head> 内でCSSファイルを読み込む
  2. セレクタが要素に一致する
  3. 文法ミスがない
  4. ブラウザを更新して確認
<link rel="stylesheet" href="styles.css" />

反映されない時のチェック

  • ファイル名のスペル（style.css と styles.css）
  • 保存忘れ
  • セレクタのタイプミス（例: .title と class="tilte"）

練習問題

  1. class="title" を付けて、CSSで色と余白を変える
  2. あえてスペルミスを作り、直して反映されることを確認する

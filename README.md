# html-css-tutorial

Mac + VSCode環境で、HTML/CSSを0から学ぶ初心者向け教材です。

## ローカルで開く

1. このリポジトリをVSCodeで開く
2. `index.html` をブラウザで開く（またはLive Serverを利用）
3. カリキュラム順に各ページを進める

## GitHub Pagesで公開する

1. GitHubの対象リポジトリを開く
2. **Settings → Pages** を開く
3. **Source** を `Deploy from a branch` に設定
4. Branchを `copilot/create-html-css-tutorial`（またはマージ後の`main`）/`root` に設定して保存
5. 表示されたURLにアクセスして公開ページを確認

公開後は `index.html` がトップページとして表示されます。
